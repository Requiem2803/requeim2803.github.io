Code Project - Page Transition

The following is a segment of code from my portfolio website. It is one of my first JavaScript scripts.
Initally it fades in a page from blank however this script truly shines when used on a link. It uses "preventDefault" to stop the link from working, stores the location of that link as a variable,
fades out the screen and then calls the link variable to transfer pages after the fadeout is completed.
By storing the link as a variable it allows the script to run regardless of the link being used, as long as it is assinged the correct class in HTML.
Finally, by hiding the original content and stopping the link using JavaScript instead of generating it within JavaScript it ensures the website still functions even in browsers where JavaScript is not available.
I feel like this brings a sense of unison to all the different pages. Creating the illusion that you never left the original page.


        // hides the main display whilst ensuring the page is still useable without javascript.
        $("main, header").css("display","none");

        // reveals the content as soon as the page is done loading.
        $(function() {
            $("main, header").fadeIn(1000);
        });

        // once a link with the class "change page" is clicked, it fades out the main and header before changing the page.
        $(".change-page").on("click", function(e) {
            e.preventDefault();
            const linkLocation = this.href;
            $("main, header").fadeOut(1000, function() {
                window.location = linkLocation;
            });
        });
